# Iris Dataset Analysis Project – Level 1

## Project Overview
This project focuses on Data Cleaning, Preparation, and Exploratory Data Analysis (EDA) using the Iris Dataset. The goal of the project is to clean the dataset, identify data quality issues, perform statistical analysis, and visualize patterns and relationships within the data using Python.

---

# Dataset Used
- Iris Dataset

The dataset contains measurements of iris flowers, including:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species Classification

---

# Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

# Project Tasks

## Task 1: Data Cleaning & Preparation

### Objectives
- Load dataset using pandas
- Identify missing values
- Handle missing values
- Detect and remove duplicate rows
- Standardize column names and text formats
- Export cleaned dataset

### Cleaning Processes Performed
- Standardized column names
- Checked dataset structure and information
- Identified duplicate rows
- Removed duplicate records
- Checked for missing/null values
- Cleaned text columns using string operations
- Saved cleaned dataset as CSV file

### Key Results
- Initial Dataset Shape: `(150, 5)`
- Duplicate Rows Removed: `3`
- Final Dataset Shape: `(147, 5)`
- Missing Values Found: `0`

# Key Insights

## 1. Species Separation
The Setosa species is clearly separated from the other flower species based on petal measurements.

## 2. Strong Correlations
Petal Length and Petal Width show strong positive correlation.

## 3. Distribution Patterns
Histograms revealed variations in feature distributions across the dataset.

## 4. Data Quality
The dataset was relatively clean with no missing values and only a few duplicate records.
